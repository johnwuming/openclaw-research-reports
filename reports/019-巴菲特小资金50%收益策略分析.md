# 巴菲特"小资金50%+收益"策略深度分析

> 研究编号：R-019 | 完成时间：2026-03-31

---

## 一、原始出处

### 核心引用

1999年，巴菲特接受 **《BusinessWeek》** 采访时说：

> *"The highest rates of return I've ever achieved were in the 1950s. I killed the Dow. You ought to see the numbers. But I was investing peanuts then. It's a huge structural advantage not to have a lot of money. I think I could make you 50% a year on $1 million. No, I know I could. I guarantee that."*
>
> — Warren Buffett, BusinessWeek, 1999

**关键细节：**
- **资金规模**：100万美元（$1 million）
- **收益率承诺**：每年50%，巴菲特强调"I guarantee that"
- **时间背景**：1950年代是他回报最高的时期，当时管理"花生米"大小的资金
- **核心逻辑**："不拥有大量资金是一个巨大的结构性优势"

### 常见误引

⚠️ 许多中文文章将此言论错误归因于1998年佛罗里达大学演讲。实际出处是1999年BusinessWeek采访。BuffettFAQ.com记录了股东年会追问巴菲特"今天是否还会这么说"，但完整回答未找到可靠原文。

> 来源：[BuffettFAQ.com](https://buffettfaq.com/)、[SchiffSovereign](https://www.schiffsovereign.com/investing/warren-buffett-its-a-huge-advantage-not-to-have-a-lot-of-money-19216/)、[雪球](https://xueqiu.com/5760376667/290584827)

## 二、巴菲特早期合伙基金战绩

### 1957-1969 合伙基金业绩

| 指标 | 数据 |
|------|------|
| 复合年收益率 | **29.5%** |
| 最高年份（1968） | **58.8%** |
| 最低年份 | 无一年亏损 |
| 同期道琼斯年回报 | ~7.8% |
| 运作年限 | 13年 |

> 来源：[Net Net Hunter](https://www.netnethunter.com/buffett-partnership-strategy-ideas/)

### 三大策略分类

巴菲特将合伙基金投资分为三类：

1. **Generals（低估型）**：被严重低估的股票，不谋求控制权
   - Generals-Private Owner：以私人买家视角评估的极度低估股
   - Generals-Relatively Undervalued：相对低估的股票
2. **Workouts（套利型）**：并购套利、清算、重组等事件驱动策略，提供稳定低风险回报
3. **Controls（控制型）**：大量买入取得控制权，主动释放价值

> 来源：[MOI Global - Buffett's Ground Rules](https://moiglobal.com/warren-buffetts-ground-rules/)

## 三、核心策略详解

### 策略1：烟蒂股（Cigar Butt / Net-Net）

**原理**：以低于公司清算价值的价格买入，就像捡起地上还能抽最后一口的雪茄烟蒂。

**量化标准 — NCAV（净流动资产价值）**：

```
NCAV = 总流动资产 - 全部负债

买入条件：市值 < NCAV（即 P/NCAV < 1）
更严格标准：市值 < 2/3 × NCAV（Graham原始建议）
```

**历史验证数据**：

| 市场 | 时间段 | NCAV策略年化 | 基准年化 |
|------|--------|-------------|---------|
| 美国 | 1971-1983 | **33.7%** | 12.1% |
| 美国 | 2003-2010 | **24.7%** | — |
| 伦敦 | 1980-2005 | **31.1%** | 20.5% |
| 日本 | 1975-1988 | **19.7%** | 16.6% |

> 来源：[Wikipedia - NCAV](https://en.wikipedia.org/wiki/NCAV)（引用 Oppenheimer 1986、Dudzinski & Kunkel 2014、Xiao & Arnold 2008 等学术论文）

### 策略2：控制型投资（Controls）

**经典案例：Sanborn Map（1958）**
- Sanborn Map 拥有价值约每股$65的证券投资组合
- 股价仅约$45（地图业务停滞被市场低估）
- 巴菲特买入后取得董事会控制权
- 将投资组合分配给股东，实现约50%回报

**经典案例：Dempster Mill（1958-1961）**
- 以每股$16-18买入农具制造公司
- 取得控制权后更换CEO（Harry Bottle）
- 削减成本、释放资产价值
- 股价涨至$80+

> 来源：[Deep Value Insights](https://www.deepvalueinsights.com/p/a-structurally-declining-business)、[Net Net Hunter](https://www.netnethunter.com/buffett-partnership-strategy-ideas/)

### 策略3：集中投资（区别于Graham）

**巴菲特 vs Graham 的关键分歧**：

| 维度 | Graham/Schloss | 巴菲特 |
|------|---------------|--------|
| 持仓数量 | 100+只 | 5-10只集中 |
| 选股深度 | 简单量化筛选 | 深度研究 |
| 参与方式 | 被动持有 | 主动股东行动 |
| 回报 | ~21%年化（Schloss 28年） | ~29.5%年化（合伙基金13年） |

巴菲特认为集中投资5-10只高确信度的deep value股票，而非分散持有100+只，是获得超额回报的关键。

> 来源：[Saber Capital Management](https://sabercapitalmgt.com/buffett-vs-munger-vs-schloss-and-thoughts-on-portfolio-strategy/)

### 策略4：事件驱动/套利（Workouts）

巴菲特的Workouts类别包括：
- **并购套利**：宣布收购后买入目标公司
- **清算套利**：公司宣布清算时以折价买入
- **重组套利**：分拆、资产出售等

这类策略提供与市场走势低相关的稳定回报，在熊市中尤其有价值。

## 四、为什么大资金做不到？

### 规模诅咒

1. **流动性陷阱**：微盘股日成交量极低，大资金买入本身就会推高股价
2. **强制升级**：管理数十亿的资金必须投蓝筹，进入竞争更激烈的领域
3. **信息优势丧失**：小盘股几乎没有分析师覆盖，个人投资者可建立信息优势；大盘股被成百上千人研究
4. **报告负担**：大基金持仓需公开披露，被跟风者稀释alpha

### 巴菲特自己的转型

- **转折点**：1960年代末，受Charlie Munger影响
- **标志事件**：1972年收购See's Candies
- **核心转变**：从"以便宜价格买入平庸公司"转向"以合理价格买入优秀公司"
- **原因**：资金规模增大后，烟蒂股的绝对收益不足以移动总回报的指针

> 来源：[The Investor's Podcast](https://www.theinvestorspodcast.com/articles/cigar-butt-investing/)

## 五、现在是否仍然可行？（2025-2026）

### 利好因素

1. **微盘股估值处于历史低位**：2024年初Barron's报道微盘股基金处于"超便宜"水平，为经济复苏后的反弹奠定基础
2. **现代筛选工具极大降低研究成本**：
   - GuruFocus Stock Screener（NCAV筛选）
   - MagicFormulaInvesting.com
   - guruscreener.io
   - 各类量化平台提供系统化回测
3. **信息不对称仍然存在**：微盘股（尤其是OTC/粉单市场）分析师覆盖极少
4. **NCAV策略在多个国际市场（包括中国）被验证有效**
5. **效率较低的市场（如印度、部分新兴市场）可能提供更大机会**

> 来源：[Barron's](https://www.barrons.com/articles/microcap-funds-super-cheap-big-returns-2024-95c1cd0b)、[Quant Investing](https://www.quant-investing.com/blog/magic-formula-investment-strategy-back-test)

### 挑战与现实

1. **50%年化是极端目标**：
   - Greenblatt的Magic Formula回测（2003-2015）实际年化仅11.4%（S&P 500为8.7%）
   - Greenblatt本人40%年化是在特殊时期（1985-2005）
   - NCAV策略学术验证的24-34%年化更现实，但需要严格执行
2. **微盘股流动性风险真实存在**：买卖价差大，退出困难
3. **公司治理风险**：微盘股/粉单市场存在欺诈、信息披露不完善
4. **策略拥挤度增加**：NCAV和Magic Formula已成为公开知识，可能降低未来超额收益
5. **交易成本**：小盘股的高买卖价差侵蚀利润

### 判断

**在当前环境下，小资金投资者获得20-30%年化是现实可行的（通过严格执行deep value策略），但50%年化需要特殊条件（极端低估市场、特殊事件催化、或效率极低的市场）。** 巴菲特的50%承诺应理解为方向性声明而非精确预测。

## 六、具体操作框架

### 框架A：NCAV / Net-Net 策略（Graham式）

**筛选标准**：
```
1. 市值 < 5亿美元（越小越好）
2. NCAV = 流动资产 - 总负债 > 市值（P/NCAV < 1）
3. 更严格：P/NCAV < 0.67（2/3折价）
4. 无持续经营亏损（排除濒临破产）
5. 有一定经营历史（非壳公司）
```

**组合构建**：
- 持仓数量：20-30只（分散化降低个股风险）
- 等权配置
- 持有周期：1-2年，价值回归后卖出
- 再平衡：年度审查

**风险管理**：
- 单只股票不超过组合的5-8%
- 设置最大亏损容忍度
- 关注审计报告和关联交易

### 框架B：集中Deep Value（巴菲特式）

**筛选标准**：
```
1. 市值 < 3亿美元
2. P/B < 0.8，P/E < 10
3. 存在明确催化剂（管理层变更、资产出售、分拆、回购）
4. 管理层持股比例高（利益一致）
5. 资产负债表健康（低负债率）
6. 有可理解的业务模式
```

**组合构建**：
- 持仓数量：5-10只（集中投资）
- 不等权，高确信度仓位更大（15-25%）
- 持有周期：无固定期限，直到价值充分释放
- 主动跟踪：与管理层沟通、参加股东大会

**风险管理**：
- 必须有深度研究的信心支撑集中度
- 保持20-30%现金储备等待更好的机会
- 只投自己能理解的业务

### 框架C：事件驱动 / Special Situations

**机会类型**：
```
1. 并购套利：收购价与市价之间的价差
2. 清算套利：公司宣布清算，股价低于预期分配价值
3. 分拆（Spinoffs）：母公司分拆业务，分拆后常被机构被动抛售导致低估
4. 法律/监管事件：诉讼和解、监管变更触发价值重估
5. 财务重组：债务重组后资产负债表改善
```

**组合构建**：
- 可与框架A或B组合使用
- 事件驱动部分占总仓位20-40%
- 每笔交易有明确的催化剂和时间表

| 工具 | 用途 |
|------|------|
| GuruFocus Screener | NCAV、P/B、P/E多维度筛选 |
| MagicFormulaInvesting.com | Greenblatt Magic Formula排名 |

1. **巴菲特合伙基金各年份详细回报数据**未完整获取
2. **粉单市场具体交易案例**（巴菲特在OTC市场的操作细节）较少
3. **2024-2025年Net-Net策略具体表现数据**缺失
4. **中国A股市场**NCAV策略的适用性和调整建议有待深入研究
5. **特殊事件（Special Situations）的具体操作清单**不够详细
6. 巴菲特在股东年会上被追问"今天是否仍坚持50%说法"的**完整回答**未找到
